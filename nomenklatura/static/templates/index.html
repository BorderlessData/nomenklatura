What can nomenklatura do for you? The service supports a set of useful tasks, such as:

  • Clean up misspelled and duplicate names in source data. Individuals or organisations often have multiple names used to identify them, and sometimes they're just misspelled. Nomenklatura creates links between these different names for the same thing to help you treat them as a single entity.
  • Integrate multiple datasets. Alternate names for entities in Nomenklatura can come from more than one data source, enabling you to make links between these sources based on partial overlaps of naming conventions.
  • Share your data cleaning work. Instead of making entity mappings ad-hoc while cleaning data, judgments are stored on the web so that they can be shared between people and re-used across different releases of a dataset.
  • Generate URIs and store additional metadata. Along with names, Nomenklatura can store additional attributes, such as address information, company identifiers and external references with each entity.

To get started, check you should sign in, create a dataset and upload information about entities to be cleaned and shared.
