What is a dataset?

A dataset is a logical unit of entities in nomenklatura. Examples of this could be all politicians in a country, all countries in the world or all companies involved in a particular context.

Be aware that a dataset in nomenklatura doesn't have to correspond to a single data source. Names from a variety of origins can be combined and mapped together in a single dataset to integrate the data sources.

How do I add entities to a dataset?

What is review mode?

In review mode, you are prompted to decide if an entity is an alias for another entity, whether it is invalid or whether it should remain as it is. To make this easier, each entity is compared to all others in the dataset, and the most similar entities are shown as choices for you to see if there is an obvious duplicate.

To make a change, just select the relevant option - another entity to merge with, or the invalid option, and save the result.

What does marking an entity as invalid mean?

Marking an entity as invalid is helpful when the entity was added to a dataset but doesn't actually make sense in the context of the dataset, i.e. when it is a category error.

For example, imagine importing speaker names from a parliamentary transcript into nomenklatura. Instead of being marked with a speaker name, some speeches may be marked in the transcript with a piece of pseudo-naming like SPEAKER, which doesn't actually represent a person's name.
